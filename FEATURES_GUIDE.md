# FEATURES GUIDE

## 🎯 Main Features

### 1. 📚 Belajar Mode (Learning Mode)
Learn Alquran with AI guidance

**How to Use:**
1. Select a Surah from the left panel
2. Click on an Ayah to expand details
3. Read the Arabic text and translation
4. Click "Dengarkan" to play audio
5. Ask AI about the Ayah using chat

**Features:**
- ✓ All 114 Surahs available
- ✓ Professional audio recitation
- ✓ Arabic text with translation
- ✓ Interactive learning
- ✓ Context-aware AI explanations

---

### 2. 🎤 Analisis Mode (Analysis Mode)
Record your Quran recitation and get feedback

**How to Use:**
1. Go to Analyze tab
2. Click "🎤 Rekam" to start recording
3. Recite the selected Ayah
4. Click "⏹️ Stop" to finish
5. AI analyzes and gives feedback

**Features:**
- ✓ Real-time recording
- ✓ Accuracy scoring (0-100%)
- ✓ Pronunciation feedback
- ✓ Suggestions for improvement
- ✓ Performance tracking

**Feedback Includes:**
- Accuracy percentage
- Common errors
- Correction tips
- Motivation & praise

---

### 3. 💬 Chat Mode (AI Chat)
Ask anything about the Quran

**How to Use:**
1. Go to Chat tab
2. Type your question
3. Wait for streaming response
4. Ask follow-up questions

**Example Questions:**
- "Apa makna Surah Al-Fatihah?"
- "Jelaskan tentang Hari Kiamat"
- "Apa hukum berpuasa?"
- "Siapa nabi terbaik?"
- "Bagaimana cara berbuat baik kepada orang tua?"

**Features:**
- ✓ Real-time streaming responses (like Gemini Live)
- ✓ Context from selected Surah
- ✓ Deep Islamic knowledge
- ✓ Indonesian language support
- ✓ Follow-up questions supported

## 🎓 Learning Tips

### For Beginners
1. Start with short Surahs (Al-Fatihah, An-Nas)
2. Listen to audio first
3. Read translation to understand
4. Ask AI for clarification
5. Try recording after learning

### For Intermediate
1. Focus on Tajweed rules
2. Record and analyze frequently
3. Ask about deeper meanings
4. Connect concepts between Ayahs
5. Practice longer Surahs

### For Advanced
1. Memorize Ayahs
2. Compare different interpretations
3. Study historical context
4. Practice complicated letters
5. Help others learn
