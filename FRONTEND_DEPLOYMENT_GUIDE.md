# Frontend Deployment Guide

## 📱 Frontend Deployment ke Vercel

Frontend siap untuk dideploy ke Vercel dengan backend API yang sudah ter-deploy di Vercel.

---

## 🚀 Quick Deploy Steps

### Step 1: Siapkan Repository
```bash
cd frontend
git add .
git commit -m "Prepare frontend for Vercel deployment"
git push origin main
```

### Step 2: Deploy ke Vercel
**Pilihan A: Via Vercel CLI**
```bash
npm i -g vercel
vercel
```

**Pilihan B: Via Vercel Web Dashboard**
1. Buka https://vercel.com
2. Klik "Add New Project"
3. Import repository frontend
4. Klik "Deploy"

---

## 🔧 Environment Variables

Vercel akan otomatis menggunakan `.env.production` dengan:

```env
REACT_APP_API_URL=https://quran-live-learn-backend.vercel.app/api
REACT_APP_WS_URL=wss://quran-live-learn-backend.vercel.app/ws
REACT_APP_ENV=production
```

### Jika ingin override di Vercel Dashboard:
1. Buka Project Settings → Environment Variables
2. Tambahkan:
   - `REACT_APP_API_URL` = `https://quran-live-learn-backend.vercel.app/api`
   - `REACT_APP_WS_URL` = `wss://quran-live-learn-backend.vercel.app/ws`
   - `REACT_APP_ENV` = `production`

```
frontend/
├── .env (development)

├── vercel.json (Vercel configuration)
├── package.json
├── public/
├── src/
│   ├── services/

│   └── ...

```

## 🏗️ Build & Test Locally

Sebelum deploy, test production build:

```bash

npm install

# Build production
npm run build

# Test production build locally (optional)
npm install -g serve
serve -s build -l 3000
```

## 🔗 API Endpoints (Production)

Frontend akan menggunakan endpoint ini:

```
Base URL: https://quran-live-learn-backend.vercel.app/api
WebSocket: wss://quran-live-learn-backend.vercel.app/ws
```

### Available Endpoints:
- `GET /api/quran/meta` - Metadata Al-Quran
- `GET /api/quran/surah/:number` - Get Surah
- `GET /api/quran/ayah/:surah/:ayah` - Get Ayah
- `GET /api/quran/audio/:surah` - Get Audio
- `GET /api/iqro` - IQRO Data
- `GET /api/health` - Health Check
- `WS /ws` - WebSocket for live chat

## 🐛 Troubleshooting

### CORS Issues
Backend sudah dikonfigurasi dengan CORS yang tepat di Vercel.

### WebSocket Connection Failed
- Verifikasi WSS URL menggunakan `wss://` bukan `ws://`
- Check `.env.production` memiliki URL yang benar

### API Not Responding
- Verifikasi backend sudah deployed di Vercel
- Check API URL di `.env.production`
- Test dengan: `curl https://quran-live-learn-backend.vercel.app/api/health`

### Build Failed
```bash
# Clear cache dan rebuild
rm -rf node_modules build package-lock.json
npm install
npm run build
```
